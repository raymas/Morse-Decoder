# Morse-Decoder

Pure python Morse decoder

## Getting Started

### Prerequisites

Required libraries :
* **Matplotlib 2.2.2**+
* **Numpy 1.14.2**+
* **librosa 0.6.0**+

Command for installing all dependencies :

#### Linux

```
sudo apt-get install python-pip
pip install matplotlib
pip install librosa
pip install numpy
```

### Installing

Clone the repository
```
git clone https://github.com/raymas/Morse-Decoder.git
```
